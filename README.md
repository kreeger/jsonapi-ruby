# JSONAPI rubygem

This is a Ruby interface to the Bukkit/Minecraft JSONAPI plugin.

## Installation

Add this line to your application's Gemfile:

``` ruby
gem 'jsonapi-ruby'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jsonapi-ruby
